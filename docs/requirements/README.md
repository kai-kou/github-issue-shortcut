## 決定事項（ユーザー確認済み・2026-07-10）

| 論点 | 決定 |
|------|------|
| 利用者スコープ | 最初から一般公開前提 |
| MVP 範囲 | Issue 作成のみ最短（ラベル・ショートカット起動は M2 以降）。「毎日使い始められる最小」の M1 スコープで確定 |
| ネイティブ化 | PWA → TWA ラップ路線（コードベースは 1 つ）。ただし **M4（TWA / Play 配布)は優先度低・実施判断保留** |
| ドキュメント粒度 | フルセット（本ディレクトリの 5 文書) |
| カスタムドメイン | 現時点では取得しない（workers.dev サブドメインで運用・OQ-7） |
| i18n | M1 から日本語・英語の 2 言語（NFR-13） |
| ライブラリ選定方針 | 高速性（バンドル最小・起動速度）を最優先に選定（OQ-2） |
| 収益化 | 現時点で考慮しない |
| マイルストーン優先順位 | 起票の速さを最重視。M2 の次は M3（クイック追加体験）を優先 |
| ボトムシート UI | 実現難易度のスパイク結果次第で優先度を上げる（M2 完了時に判定） |
| iOS・デスクトップ | 優先度低（動作は妨げないが最適化しない） |

## 運用

- 本ドキュメントは開発の進行・学びに応じて更新する（更新は PR 経由）
- 機能要件 ID（FR-x）・非機能要件 ID（NFR-x）は Issue から参照してトレーサビリティを保つ
